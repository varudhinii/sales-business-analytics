# Sales Business Analytics & Dashboard

## Overview
This project performs end-to-end sales data analysis on a synthetic retail dataset to extract meaningful business insights and KPIs. It includes data cleaning, exploratory data analysis (EDA), business intelligence metrics, and an interactive Streamlit dashboard.
Instagram: velvet.crumbss

## Dataset
- 1032 sales records
- Product types: cupcakes, cakes, bento cakes, tiramisu

- Payment modes: UPI, Card, Cash

## Key Analysis Performed
- Sales KPIs (total revenue, average order value, units sold)
- Product performance analysis
- Customer behavior analysis (repeat customers, top customers)
- Time-based trends (monthly & daily sales)
- Payment mode analytics
- Exploratory Data Analysis with visualizations
- Product recommendation system (collaborative filtering)
